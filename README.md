# 🧱 AEU – Brick Industry Website

> "Small Contribution To Build Your Dream"

A modern, responsive business website developed for a brick manufacturing company. It showcases products, services, and business details in a clean and user-friendly format.

---

## 🌐 Live Demo
https://aeu-three.vercel.app/

---

## 📌 About the Project

AEU (Annapurna Eent Udyog) is a website designed to represent a real-world brick manufacturing business.

The goal of this project is to:
- Promote local business online
- Showcase available products
- Provide easy contact and payment options
- Build a simple and effective user experience

---

## 🚀 Features

- Clean and responsive design
- Product showcase with details
- Image and video gallery
- Contact information section
- QR-based payment option
- Customer feedback section
- Easy navigation interface

---
